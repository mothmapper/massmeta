Vars

authorA string with authors of this modpack.
descA string desc for the modpack. Can be used for modpack verb list as description.
groupA string with group of this modpack. Choose between "Features", "Translations" and "Reverts"
iconAn icon for modpack preview,
idA string unique ID for the modpack. Used for self-cheсks, must be same as modpack name in code. /datum/modpack/ru_crayons -> "id = ru_crayons"
mod_dependsA list of your modpack's dependencies. If you use obj from another modpack - put it here.
nameA string name for the modpack. Used for looking up other modpacks in init.
visibleIs modpack visible, (for hidden modpacks, like "example" one)
